# Vending Machine

This project is a simulation of a vending machine implemented in C#. It includes a console-based application and a core project with the vending machine logic.

## Features

-   Accepts coins.
-   Dispenses products.
-   Manages inventory.
-   Provides a console-based interface for interaction.

## How to Run

1.  Navigate to the `VendingMachine/src/VendingMachine.App` directory.
2.  Build and run the project using `dotnet run`.
3.  Follow the on-screen instructions to interact with the vending machine.

## Project Structure

-   **VendingMachine.App:** The console application for the vending machine.
-   **Domain:** Contains the core domain logic, including models for products, slots, and the vending machine itself.
-   **Infrastructure:** Includes implementations for hardware components like the coin acceptor and dispenser.
