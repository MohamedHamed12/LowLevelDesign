# Tic-Tac-Toe

This is a Tic-Tac-Toe game implemented in C# with a console-based UI. The project is structured into several layers, including application, core, and infrastructure.

## How to Play

1.  Navigate to the `TicTacToe/TicTacToe.ConsoleUI` directory.
2.  Build and run the project using `dotnet run`.
3.  The game will start, and you can make your moves by entering the row and column numbers.

## Project Structure

-   **TicTacToe.Application:** Contains the application logic, including the game controller and player implementations.
-   **TicTacToe.Core:** Defines the core entities of the game, such as the board and marks.
-   **TicTacToe.ConsoleUI:** The console user interface for the game.
-   **TicTacToe.Infrastructure:** Can be used for infrastructure-level concerns like data persistence.
-   **TicTacToe.Tests:** Includes unit tests for the game.
